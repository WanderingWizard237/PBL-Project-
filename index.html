About the Project

DRROS is a smart disaster management system designed to optimize resource allocation using real-time data. The system integrates external APIs like GDACS to fetch live disaster data and uses intelligent algorithms to allocate resources efficiently.

Key Features

  • Real-time disaster data using GDACS API
  • Optimized resource allocation system
  • Weather factor integration for better decisions
  • Interactive dashboard visualization
  • Incident severity-based prioritization

Tech Stack

  • Python (Core Logic)
  • APIs: GDACS, Weather API
  • Frontend: HTML, CSS
  • Version Control: Git & GitHub
