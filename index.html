🚀 DRROS

Disaster Response Resource Optimization System

DRROS is a real-time intelligent disaster management system designed to optimize emergency response using live data. It integrates global disaster feeds and environmental conditions to make smart decisions.

Unlike traditional systems, DRROS dynamically evaluates disaster severity and external conditions to prioritize incidents and allocate resources efficiently.

⚙️ How It Works

  1. Fetch live disaster data from GDACS
  2. Fetch real-time weather conditions
  3. Compute priority score
  4. Visualize data on dashboard and map
  5. Allocate resources dynamically
